# DinDin-Financas
>> https://dindinfinancas.netlify.app/ <<
Aplicação para controle de finanças pessoais.
Pode-se adicionar, editar, filtrar e ordenar despesas e receitas.

Tecnologias: node.js, postgres e react.js.
